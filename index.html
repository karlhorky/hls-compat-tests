Native HLS compatibility tests

This page tests native browser HLS behavior, using the HLS.js demo streams.

Native HLS compatibility varies across browsers. Compatibility problems can show up with eg. streams using gzip-encoded .m3u8 playlists (Chromium issue 507987650), audio-only, alternate audio, subtitles, captions, and live or low-latency streams.
